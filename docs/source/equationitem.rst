EquationItem
============

=======
Summary
=======

An **EquationItem** represents a `LaTeX <https://en.wikibooks.org/wiki/LaTeX/Mathematics>`_
defined math item.

Properties
----------------

**id**: :doc:`key`
  The unique identifier for the *EquationItem* item.

**latex**: *String*
  The LaTeX string that describes this math equation item.
