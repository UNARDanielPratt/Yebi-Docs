ULSvgLinePoints
===============

=======
Summary
=======

The point or points describing an svg line or curve.

Properties
-----------

**id**: :doc:`key`
  A unique identifier for this item.
  Note: This is used to help saving edited data

**point1**: :doc:`/ulpoint`
  The point to move to

**point2**: :doc:`/ulpoint` *Optional*
  Optional control point
