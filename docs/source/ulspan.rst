ULSpan
===========

=======
Summary
=======

A **ULSpan** keeps track of the start and end indexes of an item.

Declaration
-----------

.. code-block:: swift

   struct ULSpan

Properties
----------------

**id**: :doc:`key`
  The unique identifier for the **ULSpan** item.

**start**: *Int*
  The starting index of the span.

**end**: *Int*
  The ending index of the span.
